# Component Documentation

## Modal Component

Purpose:
Displays content in a popup window.

Features:

* Open/Close Modal
* Click Outside to Close
* Responsive Layout

---

## Accordion Component

Purpose:
Displays collapsible content sections.

Features:

* Expand Section
* Collapse Section
* Smooth Interaction

---

## Tabs Component

Purpose:
Allows users to switch between multiple content sections.

Features:

* Active Tab Highlight
* Dynamic Content Switching

---

## Dropdown Component

Purpose:
Provides a list of selectable options.

Features:

* Easy Selection
* User-Friendly Interface

---

## Toast Notification Component

Purpose:
Displays temporary feedback messages.

Features:

* Auto Hide
* Responsive Design

---

## State Management

JavaScript is used to manage:

* Modal Visibility State
* Active Tab State
* Toast Visibility State

---

## Technologies

* HTML5
* CSS3
* JavaScript

## Conclusion

This project demonstrates reusable component development, state management, event handling, and modern frontend UI design.
